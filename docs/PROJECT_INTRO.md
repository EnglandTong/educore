# EduCore · 光合啟途

> **让每个孩子拥有平等的学习机会**

---

## 🎯 我们在做什么

EduCore（光合啟途）是一个专为乡村教育设计的 **AI 驱动自适应学习平台**。

我们相信：**教育公平，不是给每个孩子一样的东西，而是给每个孩子最适合他们的东西。**

---

## 💡 核心理念

| 原则 | 含义 |
|------|------|
| **🔬 科学自适应** | 基于 BKT / IRT / SM-2 算法，动态调整题目难度和出现频率 |
| **❤️ 温暖不评判** | 永远不说"你错了"，只说"让我们一起再想想看" — 零压力学习环境 |
| **📶 离线优先** | PWA + 本地大模型，即使没有互联网也能完整使用 |
| **🏫 全员协同** | 学生 ↔ 家长 ↔ 老师 ↔ 志愿者，一个都不能少 |

---

## 🚀 我们解决什么问题

### 乡村教育的三大痛点

1. **师资匮乏** — 一个老师带 3-5 个年级，根本顾不过来
2. **资源贫瘠** — 没有互联网、没有教辅、没有课外辅导
3. **信息断层** — 家长在外打工，完全不知道孩子的学习状况

### 我们的解法

| 痛点 | EduCore 方案 |
|------|-------------|
| 师资不够 | AI 老师 24 小时在线，个性化辅导每个孩子 |
| 没有网络 | **树莓派 5 + 本地大模型**，完全离线运行 |
| 家长失联 | 微信小程序推送每周学习报告，千里之外也能参与 |

## 🛠️ 技术特色

### 🌍 三级 AI 降级链

```
本地 Ollama (树莓派) → 云端 Ark API → 规则引擎
```

- **有网时**：云端大模型提供高质量讲解
- **弱网时**：自动切换到本地小模型（Qwen 2.5 1.8B）
- **断网时**：规则引擎兜底，保证学习不中断
